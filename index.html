Arduino IR Wand + LED Control System

Infrared-controlled lighting system with embedded microcontroller logic

Designed and built an infrared-controlled LED system using embedded microcontroller logic.

This project integrates hardware circuitry with embedded software to transmit and decode IR signals, enabling wireless control of LED patterns and lighting behaviors.

The system interprets button presses, holds, and repeat signals to trigger different color modes and power states. The hardware is currently being refined into a custom PCB design using KiCad.

AI-based karate match analysis (in progress)

Building AI detection in karate for opponent data analysis (in progress).

I am developing a computer vision system that analyzes karate match footage to track athletes, detect techniques, and generate performance data.

The system uses multi-person detection, pose estimation, and joint-angle validation to recognize techniques in real time while maintaining identity tracking across frames.
